# 🚀 Nox

Projeto full stack com frontend em Angular e backend em Java (Spring Boot).

---

## 📌 Observação Importante

Para iniciar o frontend corretamente, é necessário acessar a pasta específica do projeto:

```bash
cd C:\Users\jonathas.silva\Downloads\ProjectStudy\Nox\nox-frontend
ng serve
```

---

## 🅰️ Versão do Angular

Este projeto utiliza:

```
Angular 17
```

---

## ⚙️ Configuração do Node (Obrigatório)

Para garantir compatibilidade com o Angular 17, utilize o Node.js versão 20.

### 👉 Comando para forçar o uso do Node 20:

```bash
set PATH=C:\Users\jonathas.silva\Downloads\node\node-v20.20.2-win-x64;%PATH%
```

---

## 📦 Instalação do Angular CLI 17

Caso precise instalar o Angular 17:

```bash
npx @angular/cli@17 new nox-frontend
```

---

## ▶️ Como rodar o projeto

### 🔹 1. Acesse a pasta do frontend

```bash
cd nox-frontend
```

---

### 🔹 2. Instale as dependências

```bash
npm install
```

---

### 🔹 3. Inicie a aplicação

```bash
ng serve
```

---

### 🔹 4. Acesse no navegador

```
http://localhost:4200
```
